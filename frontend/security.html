Account Ownership Verification

To prevent fraud and the sale of stolen accounts, we require sellers to prove they own the account they are listing. This is a critical step in maintaining a trustworthy marketplace.

Mandatory

Initial Verification

This is the required first step for all sellers to list an account.

Code-Based Verification

The platform generates a unique, random code (e.g., `AccSwap-A8B2C`). The seller must post this code in the bio, a post, or a story of the social media account they are selling.

Automated Check

The backend will then attempt to scrape the public profile of the account to find this code. The listing is only approved if the code is found.

Optional but Encouraged

Enhanced Verification

Sellers can complete this step to further increase buyer trust.

Analytics Screenshots

For platforms like Instagram or YouTube, require sellers to upload recent, un-cropped screenshots of their analytics/demographics page. This helps verify follower quality and engagement claims.
